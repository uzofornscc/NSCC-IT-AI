# The Project Objective

The objective is to develop a professional portfolio and a streamlined resume designed for efficient retrieval and accessibility. Furthermore, it aims to synthesize the knowledge gained from the IT Programming for AI curriculum and provide a granular breakdown of specialized technical competencies.


## The Resume

An automated resume will be produced that can be readily updated without reworking the format.

## The Structure of IT Programming for AI

The skills acquired from this program in NSCC are of four categories:

1. Data: Skills required for processing and using datasets
2. Machine learning: Ability to derive a model from data
3. Communication: Able to present your thoughts to others.
4. Tools: Programming and Algorithms, python modules like numpy, pandas, matplotlib, scikit-learn, pytorch.
